=== Linkextractor ===
Tags: links, extractor, list, print version
Requires at least: 2.0.0
Tested up to: 2.7.1
Stable tag: 0.5

This plugin extract the links in your post and lists them under your post. Your links in the post will not be removed.

== Description ==

This plugin print all links in a post at the bottom of the post. This is usefull when you would like to have a print version, 
so without this plugin you see on the paper only the underlined link but, you don't know where the link is going to.

The linkextractor solv this problem. After installing the plugin it will print out the links at the bottom and also on your printed version.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `linkextractor.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy!
